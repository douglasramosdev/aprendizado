\chapter{Desenvolvimento}
    As tecnologias que foram utilizadas neste trabalho incluem:
\section{Frontend}
\begin{enumerate}
    \subsection{React.js}
    \item React é um framework JavaScript criado pelo Facebook (atual Meta) que é usado para criar interfaces de usuário (UI) em aplicativos web.
    Ele é popular por ser fácil de usar, altamente flexível e escalável, e é usado por muitas empresas de tecnologia, incluindo o Facebook, Instagram e Airbnb.
\end{enumerate}

\section{Backend}
\begin{enumerate}
    \subsection{Node JS e Express}
    \item Como um ambiente de execução JavaScript assíncrono orientado a eventos, 
    o Node.js é projetado para criar aplicações de rede escaláveis. Ele é leve e eficiente, 
    perfeito para aplicações em tempo real que exigem alta performance e baixa latência.
    Já o Express.js é um framework web minimalista para Node.js que facilita a criação de servidores web e APIs.
    Ele fornece uma série de recursos robustos para desenvolvimento web, como roteamento, middleware e suporte a templates.
\end{enumerate}
